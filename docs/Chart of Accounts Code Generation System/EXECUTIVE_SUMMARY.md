## 🔢 CODE NUMBERING SYSTEM

**Hierarchical, accounting-standard format:**

```
Level 1 (Root Groups)
├─ 1000 = Assets
├─ 2000 = Liabilities
├─ 3000 = Equity
├─ 4000 = Income
└─ 5000 = Expenses

Level 2 (Sub-groups)
├─ 1001, 1002, ..., 1099 (under Assets)
├─ 2001, 2002, ..., 2099 (under Liabilities)
└─ etc.

Level 3 (Accounts)
├─ 1001-001, 1001-002, ... (under group 1001)
├─ 1002-001, 1002-002, ... (under group 1002)
└─ etc.
```

## 💡 Key Concepts

### Voucher Number Components
```
[Prefix]  = Custom code (S, JV, RV, etc.)
[BranchId] = Auto from session (1, 2, 3, etc.)
[Date]    = Optional date (20260713, 2026-07-13, etc.)
[Counter] = Sequential number (0001, 0002, etc.)
```

### Date Format Placeholders
```
YYYY = 4-digit year (2026)
YY   = 2-digit year (26)
MM   = 2-digit month (07)
DD   = 2-digit day (13)
```

### Reset Policies
```
Daily           = Counter resets each day
Annually        = Counter resets each year
Never           = Counter never resets
Per Financial Yr = Counter resets on FY date
```
